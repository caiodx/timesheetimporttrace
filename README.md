# TimesheetImportTrace

Aplicação **Electron + React + TypeScript + Redux + Material UI**.

## Scripts principais

- `npm run dev` – inicia Electron em modo desenvolvimento com o renderer React via Vite.
- `npm run build` – gera build do renderer e do processo principal.
- `npm run lint` – roda ESLint nos arquivos TypeScript/TSX.

## Passos iniciais

1. Instalar dependências:

   ```bash
   npm install
   ```

2. Rodar em desenvolvimento:

   ```bash
   npm run dev
   ```

> Observação: se o `npm install` acusar arquivos bloqueados no Windows (EBUSY/ENOTEMPTY), feche quaisquer terminais/IDE que possam estar usando `node_modules` e tente novamente.
